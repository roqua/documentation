# Vragenlijsten afnemen

Bovenaan elk scherm in RoQua staat de knop "Nieuwe Meting".

<Screenshot src="/screenshots/epd/measure-page.png" />

Onderin staat de lijst van alle protocollen die binnen jouw organisatie geconfigureerd zijn. Binnen een protocol kunnen twee dingen zitten:

* Handmatige metingen: dit is waar je eenmalig met de hand vragenlijsten kiest en klaarzet. Tijdens het klaarzetten kun je kiezen voor een uitnodiging om te versturen, en eventueel een herinnering.

## Zoeken en filteren

<Screenshot src="/screenshots/epd/measure-filters.png" />

Om snel het juiste protocol te vinden kun je de lijst filteren op verschillende manieren:

### Afdeling

Met dit filter kun je de protocollen beperken tot een specifieke afdeling. Deze instelling slaat op naar je gebruikersprofiel en wordt ook op andere pagina's binnen RoQua gebruikt.

### Type meting

Hier kun je filteren op het soort meting:
- **Alle**: toont alle beschikbare protocollen
- **Autoprotocollen**: alleen protocollen die automatisch vragenlijsten klaarzetten
- **Handmatige metingen**: protocollen waarbij je handmatig vragenlijsten kiest en klaarzet
- **Expertmodules**: speciale modules zoals bijvoorbeeld PETRA

### Vrije tekst zoeken

Je kunt ook zoeken op tekst. Het systeem doorzoekt dan de volgende velden:
- Protocolnaam
- Metingnaam
- Vragenlijstnaam

Als je zoekt op tekst gebeuren er twee dingen:
- Alle protocollen klappen automatisch open, zodat je meteen kunt zien wat erin zit
- Als een vragenlijstnaam matcht met je zoekopdracht, dan tonen we een lijstje van de vragenlijsten die gevonden zijn. Zo weet je precies waarom een bepaald protocol in de resultaten verschijnt.
